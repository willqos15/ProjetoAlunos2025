# Projetos dos Alunos — Robótica Educacional

Este repositório reúne os **sites desenvolvidos por alunos do 9º ano** em 2025 durante as aulas de **Robótica Educacional** do **Colégio Objetivo Sapiens - Altamira Pará**.

O projeto teve como foco a aplicação prática de **HTML e CSS**, aliando fundamentos técnicos à criatividade individual de cada aluno.

---
### Link de acesso
(https://willqos15.github.io/ProjetoAlunos2025/)

---

## Proposta do Projeto

A ideia central foi permitir que cada aluno **escolhesse livremente o tema** do seu site, incentivando:
- Criatividade
- Autonomia
- Identidade visual própria
- Interesse genuíno pelo desenvolvimento web

Em vez de seguir um modelo único, os alunos foram estimulados a criar projetos que representassem seus gostos pessoais, como jogos, animais, música, cultura pop e outros temas de interesse.

---

## Organização do Site

A página inicial do projeto funciona como um **hub de acesso**, contendo:

- Botões com o **nome de cada aluno**, direcionando para seu respectivo site
- Um botão adicional para **download de todos os projetos**, disponível via Google Drive

Essa estrutura facilita a navegação, a visualização individual dos trabalhos e o compartilhamento do material completo.

---

## Conteúdo Desenvolvido

Durante as aulas, os alunos trabalharam conceitos como:

- Estrutura básica de HTML
- Uso de tags semânticas
- Estilização com CSS
- Cores, fontes e organização visual
- Criação de layouts simples
- Responsividade básica
- Organização de arquivos

Todo o conteúdo foi produzido **integralmente pelos alunos**, com acompanhamento e orientação em sala de aula.

---

## Objetivo Educacional

Mais do que aprender código, o objetivo do projeto foi:

- Desenvolver pensamento lógico
- Estimular criatividade e expressão pessoal
- Trabalhar responsabilidade com projetos próprios
- Apresentar o desenvolvimento web como ferramenta de criação

Cada site representa não apenas um exercício técnico, mas também o **processo de aprendizado e evolução** de cada aluno.

---

## Observações

- Os projetos foram desenvolvidos exclusivamente com **HTML e CSS**
- O foco não foi a perfeição técnica, mas a **experimentação e o aprendizado**
- O repositório funciona como um **registro pedagógico** e um **portfólio coletivo** da turma
